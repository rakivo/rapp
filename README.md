# [rapp](https://github.com/rakivo/rapp/tree/master)

# Quick start
> if you're not on X11, the program won't compile / work for you.
```console
$ rush -t release
```
> if you're not using [rush](https://github.com/rakivo/rush) ... kinda nah but ok then do `./build.sh`

> run it
```console
$ ./build/rapp-release
```
